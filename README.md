Automated tests
===============

Preparations
------------

Make sure that cordova can run the app: `cordova run ios`

1. Start the `appium` server while in this folder (install with: `npm install -g appiium; appium`)
2. `npm install -g mocha`
3. `npm install`


Running the tests
-----------------

1. `mocha test.js`
